## 核心设计理念

### 肉鸽三阶段成长体验
```
前期（波次1-4）：少而精 → 练连击 → 感受"我变强了"
中期（波次5-8）：多而杂 → 压迫感 → 逼你尽快构筑  
后期（波次9-10）：多而脆 → 割草爽 → 验证构筑
```

### 关键原则
- **小怪血量固定**：山魈300HP，毕方200HP（极致跨度感来源）
- **玩家指数成长**：通过三选一让伤害从"打不动"到"秒一片"
- **数量渐进压迫**：从3个逐步增加到15个+BOSS

### 伤害公式
```
实际伤害 = 基础攻击 × (1 + 连击数 × 连击加成率)
```

## 体验曲线

```
波次1-2：3个怪，手忙脚乱练5连击
波次3-4：4个怪，开始掌控节奏，3-4连击清怪
波次5-6：5-6个怪，2连击秒怪，有爽感
波次7-8：8-10个怪，1击秒，满屏割草
波次10：15个怪+BOSS，割草后集中打BOSS
```

## 墙壁弹性参数

```gdscript
# unit_base.gd 物理材质
bounce = 0.95      # 几乎不损失速度
friction = 0.0     # 无摩擦
linear_damp = 0.3  # 自然衰减很慢
```

**效果**：一次弹射可反弹 8-12 次，轻松达成高连击
