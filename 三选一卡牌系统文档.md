# 山海弹珠 - 三选一卡牌系统文档

> 版本: MVP v0.1 | 更新日期: 2026-04-07

---

## 一、卡牌类型概览

三选一系统包含 **4 种卡牌类型**：

| 类型 | 英文名 | 出现条件 | 说明 |
|------|--------|----------|------|
| 随从招募 | `RECRUIT_FOLLOWER` | 随从数量 < 3 | 招募新随从加入队伍 |
| 随从替换 | `REPLACE_FOLLOWER` | 随从数量 >= 3 | 替换现有随从 |
| 技能升级 | `UPGRADE_SKILL` | 无限制 | 为角色添加主动/被动技能 |
| 全队增益 | `GLOBAL_BUFF` | 无限制 | 全队属性加成或特殊效果 |

---

## 二、类型权重分配

### 2.1 随从未满时（< 3个）
```
招募随从: 35%
技能升级: 35%
全队增益: 30%
```

### 2.2 随从已满时（>= 3个）
```
替换随从: 15%
技能升级: 50%
全队增益: 35%
```

## 六、互斥与过滤规则

### 6.1 类型互斥
```
┌─────────────────────────────────────────┐
│  随从数量 < 3        随从数量 >= 3       │
│       │                   │             │
│       ▼                   ▼             │
│  招募随从            替换随从           │
│  (RECRUIT)          (REPLACE)          │
└─────────────────────────────────────────┘
```

### 6.2 增益卡过滤规则

| 规则 | 说明 |
|------|------|
| **随从满过滤** | `召唤随从`在随从满3个时不出现 |
| **HP危急过滤** | `治愈`、`吸血光环`仅在HP<40%时出现 |
| **唯一过滤** | 标记`unique`的卡牌已获得则不再出现 |
| **升级链过滤** | 需先获得前置技能才解锁高级版本 |
| **叠加限制** | 标记`stackable`的卡牌最多叠加3层 |
| **阶段过滤** | 根据玩家等级限制可抽取的卡牌阶段 |

## 八、品质系统

### 8.1 品质等级

| 品质 | 颜色 | 基础倍率 | BOSS掉落权重 |
|------|------|----------|--------------|
| 普通 (common) | 灰白 | 1.0x | 15% |
| 精良 (uncommon) | 绿色 | 1.3x | 25% |
| 稀有 (rare) | 蓝色 | 1.6x | 35% |
| 史诗 (epic) | 紫色 | 2.0x | 20% |
| 传说 (legendary) | 金色 | 2.5x | 5% |

### 8.2 品质影响
- 随从招募：品质影响随从属性倍率
- 技能升级：品质影响技能效果强度
- 全队增益：普通增益无品质，稀有/史诗增益有更高数值

## 九、特殊机制

### 9.1 「我全都要」
- 花费生命代价获取全部三张卡牌
- 最多使用 **3 次**
- 使用次数在界面显示 `(0/3)`

### 9.2 BOSS掉落
- BOSS波次击败后触发三选一
- 品质权重向高稀有度倾斜
- 更容易出现稀有/史诗卡牌

### 9.3 经验升级触发
- 经验满时自动触发三选一
- 战斗暂停，选择后继续
